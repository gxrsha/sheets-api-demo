
# Google Sheets API demo with Express

Demo project using Express to posting to a google sheet

## Usage

There are 2 paths you can play with 
```
GET /sheet
```
```
POST /sheet
```
```
Sample of body to send on POST request
[ {
    Network: 'NETWORK_NAME',
    Pageviews: 473848
}]
```
